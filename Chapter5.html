JavaScript Functions

Functions are reusable blocks of code that perform a specific task when called.

Example 1: Add Two Numbers

function addNumbers(a, b) {
   return a + b;
}

Example 2: Greet User

function greet(name) {
   return "Hello, " + name + "!";
}

Example 3: Find Square

function square(num) {
   return num * num;
}
